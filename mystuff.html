Things I use/like/dislike

Disclaimer

These are things I personally use/like/dislike. Generally I prefer free software to proprietary software, however there will contradictions in this list I am sure. Sometimes I will use proprietary software out of pure ease, or school/work necessity. These things are also not endorsed by me, are not concretely the best, and are subject to change.

Things I use and like

  • Computer: Some sort of dell 2 in 1 (Inspiron 13-7378 I think). However, I can count on one hand the number of times I have used it in tablet mode
  • Desktop os: Manjaro
  • Desktop environment: KDE
  • Browser: Firefox
  • Editor: Vim
  • Markdown with pandoc to compile into pdf of html files
  • Zsh with oh my zsh
  • Python 3
  • Java

Things I like but don't use

  • Emacs - Similar to vim, however my only draw to it over vim is due to the work that the people over at spacemacs have done, and org mode
  • i3wm - I am sure that my workflow could be optimized. It just is not comfy and feels like a chore to use.

Things I do not like but must use

  • Intellij - I have yet to find myself happy with using a text editor for writing java code, so I am stuck using Intellij. I think it is better than Eclipse and better than a text editor, but I am not a fan overall. I think it is the best, but still not great
